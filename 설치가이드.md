## 🚀 STEP 1 — Netlify에 무료 배포 (5분)

### 1-1. Netlify 가입
1. 브라우저에서 **https://netlify.com** 접속
2. `Sign up` 클릭 → **Google 계정**으로 가입 (가장 빠름)

### 1-2. 앱 업로드
1. 로그인 후 대시보드에서 아래 문구가 보임:
   > "drag and drop your site folder here"
2. 이 `burger-pwa` **폴더 전체**를 드래그해서 거기에 드롭!
3. 10초 후 주소가 생성됨 (예: `https://amazing-burger-abc123.netlify.app`)

### 1-3. 주소 메모
생성된 주소를 메모해두세요. 이게 앱 주소입니다.

---

## 📱 STEP 2 — 안드로이드 폰에 설치

### Chrome 브라우저 (추천)
1. 폰에서 Chrome 열기
2. Netlify에서 받은 주소 입력
3. 앱 하단에 **"앱으로 설치하기"** 배너가 뜸 → `설치` 버튼 탭
4. 또는 Chrome 오른쪽 상단 `⋮` → `홈 화면에 추가`

### Samsung 브라우저
1. 주소 입력 후 접속
2. 하단 메뉴 `☰` → `페이지 추가` → `홈 화면`

---

## ✅ 설치 완료 후
- 홈화면에 🍔 햄버거 아이콘이 생김
- 탭하면 앱처럼 전체화면으로 실행됨
- **인터넷 없어도** 기본 기능 사용 가능 (오프라인 모드)
- 데이터는 폰 내부에 저장됨 (앱 삭제 전까지 유지)

---

## 🔄 앱 업데이트 방법
앱 내용을 수정하고 싶을 때:
1. `index.html` 수정
2. Netlify 대시보드 → `Deploys` → 수정된 폴더 다시 드래그 드롭
3. 폰에서 앱 열면 자동으로 최신 버전 적용

---

## ❓ 자주 묻는 질문

**Q. 여러 폰에서 같은 데이터를 쓰고 싶어요**
A. 현재는 각 폰마다 별도 저장됩니다. 데이터 관리 화면에서 Excel로 내보내기 후 다른 폰에서 가져오기 하면 됩니다.

**Q. 앱이 업데이트 안 돼요**
A. 앱 내 데이터 화면 → 아래로 스크롤 → 브라우저 새로고침, 또는 Chrome에서 사이트 캐시 삭제

**Q. 아이콘이 이상해요**
A. 처음 설치 시 기본 아이콘이 보일 수 있어요. 앱을 한 번 열었다 닫으면 정상 아이콘으로 업데이트됩니다.
